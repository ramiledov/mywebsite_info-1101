# Personal Portfolio Website

This is my personal portfolio website for the SITE 1101 course. It showcases my projects and background in information systems.

## Pages

- **Home**: Introduction and profile photo
- **About**: Background, qualifications, and activities
- **Projects**: Showcase of projects including Project 1, Project 2 (Hour of Code), and Project 3 (Robotics)

## Technologies Used

- HTML5
- CSS3 (with responsive design)
- JavaScript

## Features

- Responsive navigation bar
- Footer with social links
- Dedicated game page with an interactive JavaScript game
- Modular code structure
- Responsive design for mobile devices

## Setup

To view the website locally:

1. Clone the repository
2. Open `index.html` in a web browser

For development, use a local server like Live Server extension in VS Code.
